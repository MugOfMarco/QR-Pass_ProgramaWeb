Acceso al Sistema
En caso de ser vigilante, ir con tu administrador y pedirle que cambie tu contraseña. En caso de ser administrador, presionar en “Olvidé mi contraseña”; se te pedirá ingresar el correo que registraste con el usuario de administrador. Al mismo le llegará un correo para que pueda cambiar su contraseña.
Verificar que las credenciales estén escritas correctamente. Si el sistema sigue denegando el acceso, contactar al administrador principal para revisar el estado de la cuenta en la base de datos.
Gestión de Usuarios (Solo Administradores)
Desde el módulo de gestión de usuarios, seleccionar la opción para agregar un nuevo registro. Deberás ingresar los datos requeridos (como nombre y número de boleta) y asignar el rol correspondiente para que el sistema lo guarde en la base de datos.
Sí. En la tabla de usuarios registrados, los administradores tienen las opciones de editar la información de un usuario existente o eliminarlo definitivamente del sistema en caso de que ya no requiera acceso.
Control de Entradas y Salidas (Vigilantes)
El alumno debe mostrar su código QR en la pantalla de su dispositivo. Como vigilante, asegurarse de que el lector escanee el código; el sistema validará la información automáticamente en la base de datos y registrará la hora exacta del movimiento.
Esto ocurre si el código no pertenece a un usuario registrado en la base de datos o si hay un problema con la lectura. Pedir al alumno que muestre el QR claramente; si el error persiste, el alumno deberá revisar su registro con la administración.
Consulta de Historial y Reportes
Ingresar al módulo de historial o registros. Allí podrás visualizar la tabla con los datos de las entradas y salidas de los alumnos, incluyendo la fecha y hora exacta en la que el sistema escaneó su código QR.
Sí, en la sección del historial de accesos se puede utilizar la función de búsqueda ingresando el número de boleta del alumno para filtrar la tabla y ver únicamente sus movimientos.
